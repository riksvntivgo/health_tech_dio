# health_tech_dio
Pequeno projeto utilizando IF, ELSE e ELIF em Python. Simulando uma assistente virtual de uma clinica Health Tech.
# Desafio
Em uma clínica HealthTech, a assistente virtual Vita ajuda pacientes a monitorar sinais vitais básicos. Para garantir um atendimento ágil, Vita precisa analisar rapidamente a temperatura corporal informada pelo paciente e dar uma orientação inicial. Se a temperatura for menor que 36.0, Vita deve alertar sobre hipotermia. Se estiver entre 36.0 e 37.5 (inclusive), ela tranquiliza o paciente dizendo que está normal. Se for maior que 37.5, Vita recomenda procurar um médico devido à febre. Sua tarefa é programar a lógica de Vita para responder corretamente a cada situação, utilizando estruturas condicionais simples.

Implemente um programa que leia uma linha contendo a temperatura corporal do paciente (em graus Celsius, com ponto decimal) e imprima a orientação adequada de acordo com as regras acima. Não utilize bibliotecas externas. Considere apenas valores válidos de temperatura.

Entrada
Uma única linha contendo um número decimal representando a temperatura corporal em graus Celsius. O valor pode ter até uma casa decimal.

Saída
Uma única linha com a orientação: "Hypothermia", "Normal", ou "Fever", de acordo com a faixa da temperatura informada.
